Aprendendo Git

GitHub é um sistema de servidor para armazenar projetos na web, que todos possam ter acesso!!

Comandos Básicos do Git

  • git init: Inicializa um repositório Git vazio.
  • git clone: Clona um repositório
  • git add: Adiciona arquivos ao índice (staging area).
  • git commit: Registra as alterações no repositório.
  • git status: Mostra o estado atual do repositório.
  • git push: Envia as alterações para o repositório remoto.
  • git pull: Puxa as alterações do repositório remoto.
  • git branch: Lista, cria ou exclui branches.
  • git checkout: Muda para um branch específico.
  • git merge: Mescla branches.
  • git log: Exibe o histórico de commits.
  • git remote: Gerencia repositórios remotos.
  • git reset: Desfaz alterações no repositório.
  • git stash: Armazena temporariamente alterações não commitadas.
  • git tag: Marca pontos específicos na história do repositório.
